Kebijakan Privasi

Terakhir diperbarui: 22 Mei 2026

KasirKu ("kami", "aplikasi", atau "layanan") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data pribadi Anda.

1. Data yang Dikumpulkan

Akun & Autentikasi: Alamat email dan nama toko saat pendaftaran.

Data Transaksi: Data produk, penjualan, pelanggan, hutang, dan laporan yang Anda masukkan ke dalam aplikasi.

Data Perangkat: Informasi perangkat (model, sistem operasi) untuk keperluan analitik dan iklan.

ID Iklan: Identifier iklan untuk menampilkan iklan melalui Google AdMob.

2. Penggunaan Data

Data yang kami kumpulkan digunakan untuk:

  • Menyediakan dan mengoperasikan aplikasi KasirKu
  • Autentikasi dan keamanan akun Anda
  • Menyimpan data bisnis Anda (produk, transaksi, laporan)
  • Menampilkan iklan melalui Google AdMob
  • Memberikan dukungan teknis
  • Mengirim pembaruan dan notifikasi terkait layanan

3. Penyimpanan Data

Data Anda disimpan secara aman di Google Cloud Firestore (Firebase). Setiap pengguna hanya dapat mengakses data miliknya sendiri, dijamin oleh Firebase Security Rules yang mengisolasi data per pengguna berdasarkan UID autentikasi.

Data lokal seperti preferensi tema dan bahasa disimpan di perangkat Anda melalui SharedPreferences.

4. Izin Perangkat

KasirKu meminta izin berikut untuk menjalankan fungsinya:

  • Kamera: Untuk memindai barcode produk
  • Bluetooth: Untuk menghubungkan ke printer termal Bluetooth
  • Penyimpanan: Untuk mengunggah foto produk dan bukti pembayaran hutang
  • Internet: Untuk sinkronisasi data dengan Firebase

5. Layanan Pihak Ketiga

Kami menggunakan layanan pihak ketiga berikut:

  • Firebase (Google): Autentikasi, database, App Check
  • Google Sign-In: Login dengan akun Google
  • Google AdMob: Menampilkan iklan
  • Google Fonts: Menyediakan font untuk antarmuka aplikasi

Setiap layanan pihak ketiga memiliki kebijakan privasi masing-masing. Kami sarankan Anda membaca Kebijakan Privasi Google.

6. Keamanan Data

Kami menerapkan langkah-langkah keamanan untuk melindungi data Anda, termasuk:

  • Firebase Security Rules yang ketat (isolasi data per pengguna)
  • Firebase App Check untuk mencegah akses tidak sah
  • Autentikasi wajib untuk mengakses data
  • Enkripsi data dalam transit (HTTPS)

7. Hak Pengguna

Anda memiliki hak untuk:

  • Mengakses: Melihat semua data yang kami simpan tentang Anda
  • Memperbaiki: Mengubah data akun dan profil toko Anda
  • Menghapus: Menghapus akun dan data Anda kapan saja
  • Ekspor: Mengekspor data Anda ke file JSON (fitur backup)
  • Menarik izin: Menonaktifkan izin perangkat melalui pengaturan sistem

Privacy Policy

Last updated: May 22, 2026

KasirKu ("we", "the app", or "service") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your personal data.

1. Data We Collect

Account & Authentication: Email address and store name during registration.

Transaction Data: Product, sales, customer, debt, and report data you enter into the app.

Device Data: Device information (model, OS) for analytics and advertising.

Ad ID: Advertising identifier for displaying ads via Google AdMob.

2. How We Use Your Data

We use the collected data to:

  • Provide and operate the KasirKu application
  • Authenticate and secure your account
  • Store your business data (products, transactions, reports)
  • Display advertisements via Google AdMob
  • Provide technical support
  • Send updates and service-related notifications

3. Data Storage

Your data is securely stored in Google Cloud Firestore (Firebase). Each user can only access their own data, enforced by Firebase Security Rules that isolate data per user based on UID authentication.

Local data such as theme and language preferences are stored on your device via SharedPreferences.

4. Device Permissions

KasirKu requires the following permissions to function:

  • Camera: To scan product barcodes
  • Bluetooth: To connect to Bluetooth thermal printers
  • Storage: To upload product photos and debt payment evidence
  • Internet: To sync data with Firebase

5. Third-Party Services

We use the following third-party services:

  • Firebase (Google): Authentication, database, App Check
  • Google Sign-In: Login with Google account
  • Google AdMob: Display advertisements
  • Google Fonts: Provide fonts for the application interface

Each third-party service has its own privacy policy. We recommend you review Google's Privacy Policy.

6. Data Security

We implement security measures to protect your data, including:

  • Strict Firebase Security Rules (per-user data isolation)
  • Firebase App Check to prevent unauthorized access
  • Mandatory authentication to access data
  • Encryption of data in transit (HTTPS)

7. Your Rights

You have the right to:

  • Access: View all data we store about you
  • Correct: Update your account and store profile data
  • Delete: Delete your account and data at any time
  • Export: Export your data to a JSON file (backup feature)
  • Revoke permissions: Disable device permissions through system settings
